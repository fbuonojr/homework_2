Portfolio

Screenshots of my projects and assignments along with a brief description of what they do and links to the GitHub repositories and deployed applications for each.


Triviatastic: Triviatastic is an application that allows users to create an account and take quizzes. If an account is created, all of the user's scores are saved to a database backed by SQL. The user can look at their previous scores, and compare their scores to other users in the database. This application uses HTML, CSS, Bootstrap, JavaScript, jQuery, MySQL, Sequelize. Link to deployed application. Link to GitHub repository.


Work Day Planner: The work day planner allows you to enter your daily tasks and save them to the local storage. The colors of the time blocks will change depending on if the time has passed, is the current hour, or is in the future. This application uses HTML, CSS, Bootstrap, JavaScript, jQuery. Link to deployed application. Link to GitHub repository.


Fitness Tracker: The fitness tracker allows you to enter your various workouts and track your exercise over time. This application uses HTML, CSS, JavaScript, MongoDB, Express. Link to deployed application. Link to GitHub repository.


Game Cafe: Game Cafe is an application that expands on Triviatastic. In addition to taking quizzes, users can play other games including Sudoku, Minesweeper, and Tic Tac Toe. Additionally, users can create chat rooms and chat with others on the site. This application uses CSS, SCSS, Bootstrap, JavaScript, React.js, MongoDB, FireBase. Link to deployed application. Link to GitHub repository.


Game Foo: Game foo is an application that is centered around video games. Users can search for information about games, look up different platforms, and watch videos of their favorite games using the Youtube API. This application uses HTML, CSS, JavaScript, jQuery, AJAX, Font Awesome, Foundation by Zurb, Bootstrap. Link to deployed application. Link to GitHub repository.
